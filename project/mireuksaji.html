Project Overview

Nov. 2021

한국화콘텐츠 확산의 보급방법이 확대됨에 따라 우리문화유산에 대한 관심이 증가되었으나,
코로나19 확산으로 직접 방문이 어려운 실정이므로, 메타버스 환경의 미륵사지를 제작하여
전 세계 MZ 세대가 쉽고 빠르게 세계유산 미륵사지를 직접 상호작용하며 체험할 수 있도록
역사문화공간을 조성하고 장기적인 발전 방향을 수립하고자 했습니다.

Objective

Visual

익산 미륵사지의 전통적 가치를 전세계의 젊은 세대에게 알리기 위해 드론 및 첨단 기술을 통한
실사 모델링과 환경 구축, 친숙한 게임적인 요소를 통해 젊은 세대들이
메타버스 환경의 문화재 속에서 소통.

Development Process

Development Process

무엇을 중점으로 제작 하였는가

인공위성 기반 데이터를 기반으로 실제 지형을 조사하고 수치화하여 지형을 제작하였고,
메타버스 환경에 맞게 최적화를 진행하여 영역을 구성하였습니다.

Key Question .2

이용자가 원하는건 무엇인가?

유저가 안정감있게 공간을 관람할 수 있도록 3차원 매핑을 진행하였고,
실사 기반의 풀과 나무, 산 등의 오브젝트들을 배치하여 생동감 있는
공간을 제작하였습니다.

현실감 문화재 고증 지형 수치화 고유 3D모델링 2차원 매핑 다채로운 색감 비대면 소통 환경 친밀감 문화유산 사람들과의 소통

Key Question .3

문화재 고증을 통한 3D 모델링

고퀄리티를 지향하는 메쉬 작업와 텍스처링을 통해 디테일이 살아있는 모델링을 제작하였고,
이를 통해 유저들이 더욱 현실감 넘치는 메타버스 공간을 체험하며, 문화유산의 가치와
아름다움을 재발견하고 친말감 형성을 이룰 수 있습니다.

- 3D environment construction through Unity game engine

- Build interactive functions through c# code

- Creation of metaverse environment through level design

- 3D mapping proceeds through field visits and satellite-based

- Real-time communication function establishment

- 3D modeling production through 3D MAX

- High-quality texture expression through Adobe Substance

- Drone shooting live-action based mesh production

- Create Unity Portable Shaders

ID Modeling

미륵사지 서탑과 동탑, 당간지주를 문화재 고증을 통해
3D 모델링으로 실사화 작업을 진행하였습니다.
